## 1. North star

v0.0.0 ships myfi as a complete, installable Claude Code finance plugin: a six-agent flock
(advisor, auditor, quant, designer, worker, and a scaffold-only trader), four skills (IMPROVE,
COMPLIANCE, TAXES, MYFI), four entry-point commands (analyze, plan, taxes, trade), and a
Python3/poetry toolkit exposed as both a CLI and an MCP server carrying numpy/pandas/scipy/matplotlib,
a per-project and an optional global SQLite database, and a market-data adapter contract. The plugin
installs, auto-discovers every component, routes all LLM calls through a local-Claude-Code service,
and produces one real financial report end-to-end via `/myfi:analyze`.

## 5. Engineering decisions (locked)

- **Layout:** components live at repo ROOT (`agents/`, `skills/`, `commands/`, `hooks/`, `bin/`,
  `.mcp.json`) beside `.claude-plugin/`. `src/` is relocated then deleted -- `src/` is not
  auto-discovered (`discovery-packaging.md` §1).
- **Toolkit:** one poetry project at `services/toolkit/` (package `myfi_toolkit`) with two entry
  points -- `myfi-toolkit` (CLI) and `myfi-mcp` (stdio MCP via `mcp.server.fastmcp.FastMCP`), fronted
  by `bin/myfi-toolkit` + `bin/myfi-mcp` wrappers (poetry-run → `python -m` fallback). MCP registered
  in root `.mcp.json` at `${CLAUDE_PLUGIN_ROOT}/bin/myfi-mcp`; venv persisted under
  `${CLAUDE_PLUGIN_DATA}` via a SessionStart hook (`discovery-packaging.md` LOCKED RECS).
- **Data science:** numpy/pandas/scipy/matplotlib pinned; matplotlib on the Agg backend so charts
  render headless to files that `@designer` consumes.
- **Databases:** per-project `.myfi/myfi.db` (git-ignored) + optional global `~/.myfi/global.db`;
  Python `sqlite3`, `0001_init.sql` + gap-fill migrations mirroring shepherd `skills/context`.
- **LLM access:** all model calls route through `services/llm/` shelling out to local Claude Code --
  NEVER a hosted API (`CLAUDE.md` §LLM access; ported from shepherd `services/llm/llm.sh`).
- **Market data:** a `MarketDataSource` adapter contract; default degrades to WebFetch/research.
  Concrete providers deferred behind the contract -- Finnhub (default), yfinance + FRED (alternates),
  Massive.com MCP (the operator's "massive market API MCP") -- see `discovery-marketdata.md`.
- **@trader is scaffold-only:** contract + cycle structure + authorization doctrine ship as docs;
  NO live order path, NO exchange wiring, NO autonomous execution in v0.0.0.
- **Adversarial pairing:** `@advisor`/`@quant` (actors) ↔ `@auditor` (adversary), REDO cap 3;
  `@designer` is the final artifact editor (ported from shepherd auditor triple + REDO loop).
- **Tests + evals:** every deliverable ships gate tests (deterministic, local, <2s) AND evals (via
  `services/eval`, scored by local Claude Code) in the same commit.
